# Sistema de Predicción IA — UPIICSA
**Autor:** Ochoa Bocker Maximiliano

## ✅ 100% GRATIS — Sin API keys, sin tarjeta de crédito

La lógica de predicción corre directamente en el navegador (JavaScript puro).
No necesitas Groq ni ninguna API externa.

### Subir a GitHub y desplegar en Render.com

1. Crea un repositorio nuevo en github.com y sube todos estos archivos
2. Ve a https://render.com → **New → Web Service**
3. Conecta tu repositorio de GitHub
4. Deja todo como está y clic en **Deploy**

¡Listo! No hay variables de entorno que configurar.

### Uso local
```bash
npm install
npm start
```
Abre http://localhost:3000
